# [觀葉森活](https://papa2415.github.io/leafAndHome/)

此專案為 **2025 六角學院 React 框架專題班 #A-1 觀葉森活** 之前端切版作品。

## 專案介紹

讓養植物這件事，不只是「買一盆回家」，而是一步一步，把生活慢慢長成你喜歡的樣子，為城市生活打造的綠色角落，簡單、安心、有質感。
